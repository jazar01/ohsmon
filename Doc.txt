To retrieve data using HTTP GET  (not yet implemented)

	Get all data for ClientID and Type
		https://ohsmon.orasi.com/api/mon/V1?clientid=<clientID>&Type=<type>

	Get all data for ClientID
		https://ohsmon.orasi.com/api/mon/V1?clientid=<clientID>

	Limit data to past n days
		add &days=<n>

	Limit data to last i records
		add &records=<i>
